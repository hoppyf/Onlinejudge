批量导入用户


请小心使用该功能，格式不规范或者多次导入会导致数据错误，修改会带来诸多不便。

限制文件:    *.xls (*.xlsx暂不支持)

格式规范:


  • 请确保是第一个sheet，放在其他sheet的数据不做处理.
  • 首行确保从1A开始，从左到右分别为 用户名、真实姓名、邮箱、密码、学号
  • 首行只是信息提示作用，非真实数据，请按上方内容进行填写(实际名字不同不会造成影响，但请保证顺序相同)
  • 接下来从2A开始，从左到右分别为和首行对应的真实数据，可以多行但请中间不要留空
  • 用户数据当中，请注意用户，邮箱是不可重复项，即有两个用户的 用户名或邮箱重复会导致注册失败
